# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in astro-securitytxt, please report it through [GitHub Security Advisories](https://github.com/dirsigler/astro-securitytxt/security/advisories/new).

Please do **not** open a public issue for security vulnerabilities.

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 0.x     | Yes       |
